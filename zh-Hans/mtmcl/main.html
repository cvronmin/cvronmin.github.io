简介

这是一个带有Metro风格的Minecraft启动器。

对于玩家

你可以使用此启动器来：
  1. 启动原版Minecraft
  2. 启动Minecraft Forge或LiteLoader
  3. 下载客户端（包括原版及Minecraft Forge）、库、资源[1]
  4. 自定义启动器

对于服务器主

你可以为启动器套用相应文件以配合你的服务器
以下是格式:

警告！

此功能尚在开发中，可能有更改或不预期的结果。

显示格式 
{
                        /*启动器标题(Unicode)*/
                        "title" : "\u6DF7\u5409Server",
                        /*服务器名称*/
                        "server-name" : "a server",
                        /*服务器IP*/
                        "server-ip" : "127.0.0.1:25565",
                        /*客户端名称(作为游戏路径)*/
                        "client-name" : ".minecraft",
                        /*是否需要下载整合包?*/
                        "need-server-pack" : true,
                        /*整合包网址(need-server-pack值为false时可留空)*/
                        "server-pack-url" : "https://www.dropbox.com/s/*****/*****?dl=1",
                        /*是否允许下载库及资源?*/
                        "allow-download-library-and-asset" : false,
                        /*是否允许重新下载库及资源?*/
                        "allow-redownload-library-and-asset" : false,
                        /*是否允许自行下载客户端?*/
                        "allow-self-download-client" : false,
                        /*是否锁定背景?*/
                        "lock-background" : true,
                        /*背景路径*/
                        "background-path" : "C:\\Users\\Ryan\\Pictures\\idle10.png",
                        /*自定义认证服务器*/
                        "auths" : [
                        {
                        /*名称*/
                        "auth-name" : "Local 1",
                        /*网址*/
                        "auth-url" : "127.0.0.1"
                        },
                        {
                        "auth-name" : "Local 2",
                        "auth-url" : "localhost"
                        },
                        {
                        "auth-name" : "Local 3",
                        "auth-url" : "127.0.0.1"
                        },
                        {
                        "auth-name" : "Local 4",
                        "auth-url" : "127.0.0.1"
                        }
                        ]
                        }
