簡介 Introduction

這是一個帶有Metro風格的Minecraft啟動器。
This is a Minecraft launcher with Metro style.

對於玩家 For Player

你可以使用此啟動器來：
You can use this launcher for:
  1. 啟動原版Minecraft
    Launch Vanilla Minecraft
  2. 啟動Minecraft Forge或LiteLoader
    Launch Minecraft Forge or LiteLoader
  3. 下載客戶端（包括原版及Minecraft Forge）、庫、資源[1]
    Download Client(including Vanilla and Minecraft Forge), libraries and resources(assets)
  4. 自定義啟動器
    Customize the launcher

對於服務器主 For Server Owner

你可以為啟動器套用相應文件以配合你的服務器
You can apply the file to match your server
以下是格式 Here is the format:

Warning!

This function is still in progress and may have unanticipated result and further changes. Use with caution.

Show the format 
{
                        /*啟動器標題 Launcher Title(Unicode)*/
                        "title" : "\u6DF7\u5409Server",
                        /*服務器名稱 Server name*/
                        "server-name" : "a server",
                        /*服務器IP Server IP*/
                        "server-ip" : "127.0.0.1:25565",
                        /*客戶端名稱 Client name(作為遊戲路徑 as game directory)*/
                        "client-name" : ".minecraft",
                        /*是否需要下載整合包 Is it necessary to download mod pack?*/
                        "need-server-pack" : true,
                        /*整合包網址(need-server-pack值為false時可留空) Mod pack address(blank is allowed if need-server-pack is
                        false)*/
                        "server-pack-url" : "https://www.dropbox.com/s/*****/*****?dl=1",
                        /*是否允許下載庫及資源Is it allow to download libraries and assets?*/
                        "allow-download-library-and-asset" : false,
                        /*是否允許重新下載庫及資源Is it allow to re-download libraries and assets?*/
                        "allow-redownload-library-and-asset" : false,
                        /*是否允許自行下載客戶端Is it allow to download client by user themselves?*/
                        "allow-self-download-client" : false,
                        /*是否鎖定背景Is it lock background?*/
                        "lock-background" : true,
                        /*背景路徑Background Path*/
                        "background-path" : "C:\\Users\\Ryan\\Pictures\\idle10.png",
                        /*自定義認證服務器custom authenticator*/
                        "auths" : [
                        {
                        /*名稱name*/
                        "auth-name" : "Local 1",
                        /*網址url*/
                        "auth-url" : "127.0.0.1"
                        },

界面 Interface

  1. 切換至遊戲啟動界面 Switch to Game Launching Interface
  2. 切換至下載界面 Switch to Download Interface
  3. 切換至設定界面 Switch to Setting Interface
  4. 切換至關於界面 Switch to About Interface
  5. 切換至高級遊戲啟動界面 Switch to Advanced Game Launching Interface
  6. 打開/關閉任務列表 Open/Close the Task List
