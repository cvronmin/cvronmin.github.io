Introduction

This is a Minecraft launcher with Metro style.

For Player

You can use this launcher for:
  1. Launch Vanilla Minecraft
  2. Launch Minecraft Forge or LiteLoader
  3. Download Client(including Vanilla and Minecraft Forge), libraries and resources(assets)[1]
  4. Customize the launcher

For Server Owner

You can apply the file to match your server
Here is the format:

Warning!

This function is still in progress and may have unanticipated result and further changes. Use with caution.

Show the format 
{
                        /*Launcher Title(Unicode)*/
                        "title" : "\u6DF7\u5409Server",
                        /*Server name*/
                        "server-name" : "a server",
                        /*Server IP*/

                        /*Client name( as game directory)*/
                        "client-name" : ".minecraft",
                        /*Is it necessary to download mod pack?*/

                        /*Mod pack address(leave it blank if need-server-pack is set to false)*/

                        /*Is it allow to download libraries and assets?*/
                        "allow-download-library-and-asset" : false,

                        "allow-redownload-library-and-asset" : false,
                        /*Is it allow to download client by user themselves?*/
                        "allow-self-download-client" : false,
