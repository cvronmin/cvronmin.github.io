簡介

這是一個帶有Metro風格的Minecraft啟動器。

對於玩家

你可以使用此啟動器來：
  1. 啟動原版Minecraft
  2. 啟動Minecraft Forge或LiteLoader
  3. 下載客戶端（包括原版及Minecraft Forge）、庫、資源[1]
  4. 自定義啟動器

對於服務器主

你可以為啟動器套用相應文件以配合你的服務器
以下是格式:

警告！

此功能尚在開發中，可能有更改或不預期的結果。

顯示格式 
{
                        /*啟動器標題(Unicode)*/
                        "title" : "\u6DF7\u5409Server",
                        /*服務器名稱*/
                        "server-name" : "a server",
                        /*服務器IP*/
                        "server-ip" : "127.0.0.1:25565",
                        /*客戶端名稱(作為遊戲路徑)*/
                        "client-name" : ".minecraft",
                        /*是否需要下載整合包?*/
                        "need-server-pack" : true,
                        /*整合包網址(need-server-pack值為false時可留空)*/
                        "server-pack-url" : "https://www.dropbox.com/s/*****/*****?dl=1",
                        /*是否允許下載庫及資源?*/
                        "allow-download-library-and-asset" : false,
                        /*是否允許重新下載庫及資源?*/
                        "allow-redownload-library-and-asset" : false,
                        /*是否允許自行下載客戶端?*/
                        "allow-self-download-client" : false,
                        /*是否鎖定背景?*/
                        "lock-background" : true,
                        /*背景路徑*/
                        "background-path" : "C:\\Users\\Ryan\\Pictures\\idle10.png",
                        /*自定義認證服務器*/
                        "auths" : [
                        {
                        /*名稱*/
                        "auth-name" : "Local 1",
                        /*網址*/
                        "auth-url" : "127.0.0.1"
                        },
                        {
                        "auth-name" : "Local 2",
                        "auth-url" : "localhost"
                        },
                        {
                        "auth-name" : "Local 3",
                        "auth-url" : "127.0.0.1"
                        },
                        {
                        "auth-name" : "Local 4",
                        "auth-url" : "127.0.0.1"
                        }
                        ]
                        }
